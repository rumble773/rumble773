## What I Work On

I build security automation for the unglamorous parts of threat research: exposure scanning, OSINT collection, attribution, infostealer triage, dark-web and forum telemetry, brand impersonation, and the pipelines that make this work repeatable.

Most of the best work is internal. It is the scanner that runs at 03:00, the enrichment job no one has to babysit, the agent that turns raw signals into analyst-ready context, and the dashboard that saves a morning before anyone knows there was a problem.

```text
raw signals
  -> collection
  -> enrichment
  -> attribution
  -> analyst-ready intel
```
